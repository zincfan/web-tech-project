ABOUT US


The Hello World project is a time-honored tradition in computer programming. It is a simple exercise that gets you started when learning something new. Let’s get started with REPO!/

You’ll learn how to:

#Create and use a repository

#Start and manage a new branch

#Make changes to a file and push them to REPO as commits

#Open and merge a pull request

What is REPO?

REPO is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

Your hello-world repository can be a place where you store ideas, resources, or even share and discuss things with others.
